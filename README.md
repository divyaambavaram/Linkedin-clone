  **  Introduction**

This project is a LinkedIn-like social media web app developed as part of the Full Stack Developer Internship Assignment given by AppDost.
It demonstrates full-stack development skills including authentication, CRUD operations, frontend design, backend APIs, and database integration.
** Tech Stack Used**
**Frontend**

React.js / Next.js

HTML, CSS, TailwindCSS / Bootstrap

Axios

**Backend**

Node.js

Express.js

MongoDB Atlas

Mongoose

JWT Authentication

Deployment

Frontend: Vercel / Netlify

Backend: Render / Railway / Vercel

Database: MongoDB Atlas

Features
 Authentication

User Signup

User Login

Password hashing

JWT-based authentication

**Post Management**

Create Post

View All Posts (Global Feed)

Post includes:

User Name

Content

Timestamp
